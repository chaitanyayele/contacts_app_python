# 📞 Contacts App

A simple and beginner-friendly **Command Line Contacts Management System** built using **Python** and **Dictionaries**.

This project allows users to manage their contacts directly from the terminal by adding, searching, updating, deleting, and viewing contacts.

---

## ✨ Features

✅ Add new contacts
✅ Search contacts by mobile number
✅ Update existing contacts
✅ Delete contacts
✅ View all saved contacts
✅ Simple menu-driven interface
✅ Beginner-friendly code structure

---

## 🛠️ Technologies Used

* Python
* Dictionary Data Structure
* Loops
* Conditional Statements
* User Input Handling
